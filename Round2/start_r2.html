Workflow of round 2

  1. Specify the images and the parameters (Detailed instructions)

  2. Launch the main window of 3D correction (Panel descriptions)

  3. Perform inspections and corrections with following operations:

    1. Localize to a cell.
    2. Correct the segmentation in slice i with following operations:
      • Merge / Delete / Division / Division-Relink / 3D Division / Insert
    3. Sort cells by sizes and Remove tiny cells.
    4. Save and load intermediate segmentation results.
    5. Export the segmentation as 2D Tiff images.
